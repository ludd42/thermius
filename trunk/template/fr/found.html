Vous avez trouvé notre ballon!

Félicitations! Bien que nous vivions sur un continent densément peuplé, les chances de trouver un petit ballon solaire sont en fait très minces! Merci de remplir le formulaire ci-dessous, afin de nous aider à localiser le lieu d'atterrissage de notre ballon.

 

Date
Veuillez entrer la date et éventuellement l'heure à laquelle vous avez trouvé le ballon.
Lieu
Où avez-vous trouvé le ballon? Entrez au moins le pays et le nom de la ville. Si vous avez les coordonnées GPS, donnez-les aussi.
Description
Quel était l'état du ballon lorsque vous l'avez trouvé? A-t-il été trouvé dans un arbre, dans une rivière? Merci de préciser les circonstances.
Si le code d'identité du ballon, mentionné sur la carte postale, est lisible, notez-le.
Contact
Comment pouvons-nous vous contacter? E-mail, numéro de téléphone, Skype, ...
